Kurulum Adımları

Sistem Gereksinimleri

PHP Sürümü: 8.2 veya üzeri

Veritabanı Kurulumu

  1. Öncelikle veritabanını oluşturun
  2. Veritabani.sql dosyasını veritabanına import edin

.env Dosya Ayarları

Veritabanı Bilgileri:

DB_HOST=localhost
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=

Site URL Ayarı:

APP_URL= site_adresi

SMTP Ayarları

SMTP bilgilerini .env dosyasında 45-52 satırları arasına giriniz.

Yönetici Ayarları

Admin URL değiştirme: (.env)

SYSTEM_ADMIN_PATH="adminsc"

Demo Modu: (.env)

SYSTEM_DEMO_MODE=true
